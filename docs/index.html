The digits of numeric sample case identifiers specify following propNav modeling options:
  • 1000's digit - Missile type; 1 = SAM, 2 = AAM
  • 100's digit - Proportional Navigation (PN) guidance method;
    1. True PN (TPN)
    2. Pure PN (PPN)
    3. Zero Effort Miss (ZEM) PN
    4. Augmented True PN (ATPN)
    5. Augmented Pure PN (APPN)
  • 10's digit - Proportional Navigation constant
  • 1's digit - Target turning acceleration in g's
